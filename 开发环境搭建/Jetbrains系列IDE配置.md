## 新建文件模板
PHP文件头

``` 
/** 
 * 项目名：  ${PROJECT_NAME}  
 * 文件名:   ${FILE_NAME}  
 * 创建者:   aguage   
 * 创建时间: ${DATE} ${TIME}  
 * 描述：    TODO  
 */
``` 
android文件头

``` 
/** 
 * 项目名：  ${PROJECT_NAME} 
 * 包名：    ${PACKAGE_NAME}
 * 文件名:   ${NAME}  
 * 创建者:   aguage   
 * 创建时间: ${DATE} ${TIME}  
 * 描述：    TODO  
 */
```
